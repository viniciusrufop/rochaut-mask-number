Documentação
[decimals]: string
Valor Defualt: ' '
Descrição: Define a quantidade máxima de números decimais
[units]: string
Valor Defualt: ' '
Descrição: Define a quantidade máxima de unidades
[allowLess]: boolean
Valor Defualt: true
Descrição: Habilita / Desabilita valores negativos
[comma]: boolean
Valor Defualt: true
Descrição: Definide a separação dos números por '.' ou ',', sendo true a separação por ','
[integer]: boolean
Valor Defualt: false
Descrição: Habilita / Desabilita números inteiros
Exemplos
Descrição: Permite positivos, negativos e separação por vírgula sem restrição de unidades ou decimais
Code: <input type="text" rochautMask />
Descrição: Permite positivos e negativos inteiros sem restrição de unidades
Code: <input type="text" rochautMask [integer]="true" />
Descrição: Permite positivos e negativos com no máximo duas casas decimais
Code: <input type="text" rochautMask decimals="2" />
Descrição: Permite positivos com duas unidades e dois decimais
Code: <input type="text" [allowLess]="false" units="2" decimals="2" />
Descrição: Permite positivos e negativos com três unidades e sepação por ponto
Code: <input type="text" rochautMask units="3" [comma]="false" />
Descrição: Permite fazer a separação de milhar por vŕgula
Code: <input type="text" rochautMask [thousandSeparator]="true" [comma]="false" decimals="2" />
Descrição: Permite fazer a separação de milhar por ponto
Code: <input type="text" rochautMask [thousandSeparator]="true" [comma]="true" decimals="2" />
